# 🚀 SAARTHI - Your Campus Guide

## 🧐 The Problem
Juniors are confused. Seniors are spammed.
There is a lack of structured guidance in college campuses.

## 💡 The Solution
Saarthi is a mentorship booking platform that filters noise and connects students directly to the right mentors.

## ✨ Key Features
- **Smart Wall:** FAQ filter to save time.
- **Golden Ticket:** 1-Message limit to ensure quality.
- **No Spam:** Structured slot booking.
